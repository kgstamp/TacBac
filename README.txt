Set Up:
- Follow the instructions under "Installing Unknown Source Apps on Samsung Devices" section to set up your Android device properly: https://www.groovypost.com/howto/map-a-network-drive-on-mac/

- Download the .apk file from TacBac's GitHub repository onto your Android device using Google Chrome Application
