Set Up:
- Install Nodejs: https://nodejs.org/en/download/
- Install Yarn: https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable
- Install Expo: https://docs.expo.dev/get-started/installation/
- Download the Expo go app on your phone
- Clone the GitHub repository to your laptop 

Make sure your laptop is on the safe wifi as your phone when demo-ing!

To demo:
- Open the project directory in your laptop's terminal
- Change your directory to the "gloveapp" directory
- Run the command "npm start" through
- Scan the qr code that is outputted in the terminal using your phone's camera
- The application will open up on your phone through the Expo Go app
